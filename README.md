# pythonProjectTemplate

## 概要

提供给其他python应用一个统一的框架，包含环境配置、数据库连接、日志、模块集成

## 依赖

1. yaml

    ```bash
    pip install pyyaml
    ```

    安装完成后，在Python代码中通过 `import yaml` 来使用YAML功能。
2. sqlalchemy

    ```bash
    pip install sqlalchemy
    ```

    安装完成后，在Python代码中通过 `import sqlalchemy` 来使用sqlalchemy功能。
