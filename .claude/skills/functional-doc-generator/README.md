# 功能文档生成器

一个智能的功能文档生成技能，能够根据当前开发上下文自动生成结构化的功能说明文档。

## 功能特性

- 🚀 **智能上下文分析**：自动识别最近修改的功能模块
- 📝 **模板驱动生成**：使用标准化模板生成规范文档
- 🎯 **精准内容提取**：从代码、配置、测试中提取关键信息
- 📋 **智能章节适配**：根据功能类型自动调整文档结构
- 🔧 **灵活输出路径**：支持自定义文档保存位置

## 快速使用

### 上下文模式（推荐）
```
"帮我生成这个功能的文档"
"刚完成了XXX功能，写个文档"
```

### 指定功能模式
```
"为XXX模块生成文档"
"帮我把支付功能整理成文档"
```

## 输出格式

生成标准的 Markdown 格式功能文档，包含：

- 文档信息和版本管理
- 功能概述和目标范围
- 详细的功能逻辑说明
- 接口定义和数据结构
- 数据库设计（如适用）
- 安全控制和权限管理
- 错误码和异常处理
- 配置项和依赖说明
- 测试验证和性能指标
- 部署运维指南
- 版本变更记录

## 智能适配

技能会根据实际代码情况自动：

- **保留相关章节**：只保留与功能相关的文档章节
- **删除无效内容**：移除不适用的模板内容
- **扩展必要信息**：添加功能特有的说明章节
- **提取技术细节**：从代码中自动提取API、模型、配置等信息

## 最佳实践

1. **确保代码注释充分**：便于提取功能信息
2. **遵循项目规范**：使用标准的目录结构和命名
3. **及时更新文档**：代码变更后重新生成文档
4. **版本化管理**：为每个重要版本生成对应文档

## 故障排查

如遇到问题，请检查：
- 项目是否使用 git 版本控制
- 代码结构是否符合 FastAPI 项目标准
- 是否有足够的注释和文档字符串
- 输出目录是否有写入权限
