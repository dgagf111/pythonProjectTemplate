# 功能文档生成器使用示例

## 示例 1：基于当前上下文生成文档

**场景**：刚完成用户认证功能的开发

**对话示例**：
```
用户：我刚完成了用户认证模块的开发，包含JWT登录、注册和密码重置功能，帮我生成文档

AI：
我来分析您的用户认证模块并生成功能文档。让我先检查一下最近的代码变更...

[分析代码后]

已为您生成完整的功能文档：docs/用户认证-v1.0.md

文档包含以下章节：
- 功能概述（JWT双令牌认证机制）
- API接口定义（登录、注册、刷新令牌）
- 数据库设计（users表、tokens表）
- 安全控制（密码加密、令牌过期机制）
- 错误码定义（1001-1010）
- 测试用例和性能指标
```

**对话示例**：
```

AI：
我来分析您的电商购物流程实现。这是一个跨多个模块的复杂功能，让我分别分析各个组件...

[分析多个文件后]

已为您生成电商购物流程功能文档：docs/购物流程-v1.0.md

文档涵盖：
- 商品浏览功能（商品模块）
- 购物车管理（购物车模块）
- 订单创建（订单模块）
- 支付处理（支付模块）
- 跨模块数据流和状态转换
- 完整的端到端测试用例
```
